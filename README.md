# MineOnline-JSON
version-info json files that are compatible with MineOnline

## How to install
**Windows**
1. Download the json files you want to add to MineOnline.
2. Place the json files in `%appdata%/.mineonline/version-info/client/`.
3. Launch MineOnline.
